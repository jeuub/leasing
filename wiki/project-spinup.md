# Развертывание проекта

## Склонировать проект

```bash
    git clone https://github.com/jeuub/leasing.git
```

## Установить зависимости

```bash
    cd leasing
    
    npm i
    # или
    yarn
```

## Запустить проект

```bash
    npm run start
    # или
    yarn start
```
